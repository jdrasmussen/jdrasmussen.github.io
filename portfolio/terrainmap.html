Shaded Relief Terrain Map

This map shows a series of landmarks, each roughly a day's hike apart along the Washington section of the Pacific Crest Trail. For this project I used a digital elevation model to create shaded relief in ArcGIS, which I combined with land cover raster data to create the base map in this graphic with custom coloring and blending in Adobe Photoshop. I then used Illustrator to add the trail and point data, exported from ArcGIS, as well as the text and background coloring. (Note: This project was exclusively focused on the map design. Do your own research before planning any backpacking trip!)
