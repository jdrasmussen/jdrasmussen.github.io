Database Web Application

I can only share a limited sample of this as it contains private data, but for this project I used HTML, java, and javascript to develop a web application that that connected to a postgres spatial database and displayed selected items using the Google Maps API. The web interface could filter loans based on the loan type, dollar amount, and origination date. With more time to devote to this project I would step beyond the level of individual datapoints to integrate analysis of patterns in the data.
